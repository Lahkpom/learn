Las flag cambian el comportamiento del comando

ls

ls -a (lista todo tipo de archivos incluyendo los ocultos)

-a es una flag

cada comando tiene sus flag

podemos verlos con el man

ls -l (los lista de forma vertical y con mas información)

tambien se puede convinar

ls -a -l

-h hace que el tamaño de los archivos sea legible para humanos

tambien se pueden poner seguidas

ls -alh


-S the upper case S, nos ordena la lista por tamaño de mayor a menor

-R nos muestra el contenido de los subdirectorios de los directorios que hay en la lista 
(y subdirectorios de los subdirectorios)
