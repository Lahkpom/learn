Modificar el contenido de los archivos sin utilizar la interfaz grafica

Esto se hace mediante editores como nano o vim

nano fileName.format

nano ruta/fileName.format

Una vez realizadas las modificaciones deseadas, presionamos ctrl + o para guardar y ctrl + x para salir

-----------

Como ver el contenido de los archivos desde la terminal?

cat

Permite visualizar

cat fileName.format

cat ruta/fileName.format

Lo imprime en la misma consola todo el contenido de golpe
