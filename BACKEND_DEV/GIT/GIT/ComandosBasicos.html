Cómo crear una carpeta la cualcontendrá al repositorio

Hay que entrar al cd "Dirección de donde se desea que esté la carpeta que contendrá al repositorio" y poner lo siguiente, esta carpeta almacenará todos los archivos

                    
                    mkdir nombre_carpeta

cómo agregar los archivos al área de preparación de mi repositorio

                    
                    git add nombre_carpeta

Cómo poner mis archivos en el repositorios

                    
                    git commit
                    (Escribir el mensaje, apretar scape)
                    :wq

                    (un atajo es:)
                    git commit -m "escribir_mensaje"

Cómo consultar la vitácora

                    
                    git log
