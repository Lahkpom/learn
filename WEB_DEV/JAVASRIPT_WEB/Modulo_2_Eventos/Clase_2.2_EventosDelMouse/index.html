click

Este es un evento que permite generar una reaccion, a través de un por ejemplo un método, al hacer click sobre el contenido de esa etiqueta, como se ve al hacer click en "Métodos". Está muy fuera de práctica poner el onclick dentro del HTML

dblclick

Solo cambiando el nombre del método se ve la diferencia.

mouseenter

Registra cada vez que el mouse entra en el área de la etiqueta

mouseleave

Este registra cada vez que el mouse sale del área de la etiqueta

contextmenu

Captura cuando hacemos click derecho, y manipularlo. Por ejemplo cuando en realidad debería mostrar un menú, podemos indicarle que no lo haga.

mousedown

registra cuando se hace una pulsacion larga sobre la etiqueta

mouseup

resgistra cuando se suelta el click luego de una pulsación larga

mousemove

registra cada vez que el mouse se mueve dentro de la etiqueta
