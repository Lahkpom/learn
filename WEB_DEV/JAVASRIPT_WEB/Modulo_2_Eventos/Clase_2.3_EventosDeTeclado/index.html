EVENTOS DEL TECLADO

Métodos

  • keydown
  • keyup
  • keypress

keydown

Reacciona cuando se presiona una tecla, y registra que tecla se usó, se puede manipular por ejemplo que presionando x o y tecla haga algo interno.

keyup

Reacciona cuando se suelta una tecla

keypress

Reacciona cuando se mantiene apretada una tecla
