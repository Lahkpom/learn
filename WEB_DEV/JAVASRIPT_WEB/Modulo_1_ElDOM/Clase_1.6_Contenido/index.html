Métodos

  • textContent
  • innerHTML
  • outerHTML

textContent

Con este método puedo capturar en un String todo el contenido que haya dentro de la etiqueta con el id que le indico, incluso si tiene hijos o etiquetas de linea, devuelve un solo Strinf (ejemplo 1)

innerHTML

Este también devuelve el contenido de la etiqueta con el id que el indico, pero me marca cuales son las etiquetas que tiene como hijo (ejemplo2)

outerHTML

Esta lo que hace es devolver el contenido completo de toda la etiqueta donde está el id que le inquemos, incluyendo la etiqueta misma y sus atributos (ejemplo3)

Extras

Estas herramientas no solo permiten leer sino también escribir dentro de las etiquetas (Ejemplo4) teniendo en cuenta euq textContent solo permita ingresar texto, mientras que los innerHTML y outerHTML permiten que escirbas etiquetas dentro y que el navegador las lea como HTML.
