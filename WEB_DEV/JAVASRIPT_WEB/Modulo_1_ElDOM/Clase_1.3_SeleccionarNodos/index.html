document.getElementById()

Sirve para obtener un elemento por el id que le indiquemos en el atributo de la etiqueta, lo podemos ver en el 1er ejemplo en .js, tener en cuenta que devuelve el elemento completo, es decir con etiquetas y todo, no solo el contenido de la etiqueta.

En el 4to Ejemplo, podemos ver que este comando sirve también para retornar una lista así como el querySelectorAll, poniendo un id al "ul" y recorrerla utilizando el querySelector dentro del getElementById.

document.querySelector() / element.querySelector()

Todo lo que es querySelector hace referencia a CSS, por lo que para poder seleccionarlos tenemos que usar el "#", y nos devuelve ese elemento. No usamos casi el querySelector. Esto se puede ver en el 2do Ejemplo.

document.querySelectorAll() / element.querySelectorAll()

Este querySelector, a diferencia del anterior, lo que nos devuelve no es un elemento, sino una colleción de elementos, como por ejemplo todo los items de una lista.
